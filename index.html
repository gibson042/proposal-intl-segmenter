    Intl.Segmenter Proposal

    1Segmenter Objects

    1.1The Intl.Segmenter Constructor

    The Segmenter constructor is a standard built-in property of the Intl object.

    1.1.1Intl.Segmenter ([ locales [ , options ]])

    When the Intl.Segmenter function is called with optional arguments the following steps are taken:

    1. If NewTarget is undefined, throw a TypeError exception.
    2. Let segmenter be ? OrdinaryCreateFromConstructor(NewTarget, "%SegmenterPrototype%", « [[InitializedSegmenter]] »).
    3. Let requestedLocales be ? CanonicalizeLocaleList(locales).
    4. If options is undefined, then
      1. Let options be ObjectCreate(null).
    5. Else
      1. Let options be ? ToObject(options).
    6. Let opt be a new Record.
    7. Let matcher be ? GetOption(options, "localeMatcher", "string", « "lookup", "best fit" », "best fit").
    8. Set opt.[[localeMatcher]] to matcher.
    9. Let lineBreakStyle be ? GetOption(options, "lineBreakStyle", "string", « "strict", "normal", "loose" », "normal").
    10. Set opt.[[lb]] to lineBreakStyle.
    11. Let r be ResolveLocale(%Segmenter%.[[AvailableLocales]], requestedLocales, opt, %Segmenter%.[[RelevantExtensionKeys]]).
    12. Set segmenter.[[Locale]] to the value of r.[[Locale]].
    13. Let granularity be ? GetOption(options, "granularity", "string", « "grapheme", "word", "sentence", "line" », "grapheme").
    14. Set segmenter.[[SegmenterGranularity]] to granularity.
    15. If granularity is "line",
      1. Set segmenter.[[SegmenterLineBreakStyle]] to r.[[lb]].
    16. Return segmenter.

    1.2Properties of the Intl.Segmenter Constructor

    The Intl.Segmenter constructor has the following properties:

    1.2.1Intl.Segmenter.prototype

    The value of Intl.Segmenter.prototype is %SegmenterPrototype%.

    This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }.

    1.2.2Intl.Segmenter.supportedLocalesOf ( locales [, options ])

    When the supportedLocalesOf method of %Segmenter% is called, the following steps are taken:

    1. Let availableLocales be %Segmenter%.[[AvailableLocales]].
    2. Let requestedLocales be ? CanonicalizeLocaleList(locales).
    3. Return ? SupportedLocales(availableLocales, requestedLocales, options).

    1.2.3Internal slots

    The value of the [[AvailableLocales]] internal slot is implementation defined within the constraints described in 1.4.

    The value of the [[RelevantExtensionKeys]] internal slot is « "lb" ».

    Note
    CLDR defines several extension keys, but this specification does not expose them.

    1.3Properties of the Intl.Segmenter Prototype Object

    The Intl.Segmenter prototype object, referred to as %SegmenterPrototype%, is itself an ordinary object. It is not a Segmenter instance, and has no instance-like internal slots.

    1.3.1Intl.Segmenter.prototype.constructor

    The initial value of Intl.Segmenter.prototype.constructor is %Segmenter%.

    1.3.2Intl.Segmenter.prototype[ @@toStringTag ]

    The initial value of the @@toStringTag property is the string value "Intl.Segmenter".

    This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: true }.

    1.3.3Intl.Segmenter.prototype.segment( string )

    When the Intl.Segmenter.prototype.segment is called with an argument string, the following steps are taken:

    1. Let segment be this value.
    2. If Type(segment) is not Object or segment does not have an [[InitializedSegmenter]] internal slot, throw a TypeError exception.
    3. Let string be ? ToString(string).
    4. Return ? CreateSegmentIterator(segment, string).

    1.3.4Intl.Segmenter.prototype.resolvedOptions ()

    This function provides access to the locale and options computed during initialization of the object.

    1. Let pr be the this value.
    2. If Type(pr) is not Object or pr does not have an [[InitializedSegmenter]] internal slot, throw a TypeError exception.
    3. Let options be ! ObjectCreate(%ObjectPrototype%).
    4. For each row of Table 1, except the header row, in table order, do
      1. Let p be the Property value of the current row.
      2. Let v be the value of pr's internal slot whose name is the Internal Slot value of the current row.
      3. If v is not undefined, then
        1. Perform ! CreateDataPropertyOrThrow(options, p, v).
    5. Return options.

    1.4Internal slots of Intl.Segmenter Instances

    Objects that have been successfully initialized as a Segmenter object also have several internal slots that are computed by the constructor:

    • [[Locale]] is a String value with the language tag of the locale whose localization is used by the segmenter.
    • [[SegmenterGranularity]] is one of the String values "grapheme", "word", "sentence", or "line", identifying the segmenter used.
    • [[SegmenterLineBreakStyle]] is present on Segmenter instances with line granularity. If present, it may be "strict", "normal", or "loose".

    1.5Segment Iterators

    The Intl.Segment.prototype.segment method returns iterators over the segments for a particular string. This section describes those iterator objects.

    1.5.1CreateSegmentIterator ( segmenter, string )

    When the abstract operation CreateSegmentIterator is called with Segmenter segmenter and string string, the following steps are taken:

    1. Let iterator be ObjectCreate(%SegmentIteratorPrototype%).
    2. Let iterator.[[SegmentIteratorSegmenter]] be segmenter.
    3. Let iterator.[[SegmentIteratorString]] be string.
    4. Let iterator.[[SegmentIteratorPosition]] be 0.
    5. Let iterator.[[SegmentIteratorBreakType]] be undefined.

    1.5.2AdvanceSegmentIterator ( iterator, direction )

    When the abstract operation AdvanceSegmentIterator is called with a segment iterator iterator and a direction either forwards or backwards direction, the following steps are taken:

    1. Let segmenter be iterator.[[SegmentIteratorSegmenter]].
    2. Let string be iterator.[[SegmentIteratorString]].
    3. Let position be iterator.[[SegmentIteratorPosition]].
    4. If direction is forwards and position is the length of string, or if direction is backwards and position is 0, return true.
    5. Find the next or previous (based on direction) break in string from the position position based on the locale, granularity and lineBreakStyle in segmenter.[[Locale]], segmenter.[[SegmenterGranularity]] and, if present, segmenter.[[SegmenterLineBreakStyle]].
    6. Set iterator.[[SegmentIteratorBreakType]] to a value representing the type of break found, using one of the values found in the table Table 2, or undefined if the boundaries of the string are reached, or if there is no meaningful type for the granularity.
    7. Set iterator.[[SegmentIteratorPosition]] to the position of the newly found break point.
    8. Return false.
    Note
    This specification does not require a specific algorithm, but a high-quality line breaking algorithm can be found in UTS 14, while an algorithm for other breaks is specified in UTS 29. Intl.Segmenter implementations which are inside of web browsers are encouraged to use the same line breaking algorithm as is used for text layout.

    1.5.3%SegmentIteratorPrototype%

    The %SegmentIteratorPrototype% object:
    • is the prototype of all segment iterators.
    • is an ordinary object.
    • has a [[Prototype]] internal slot whose value is the intrinsic object %IteratorPrototype%.
    • has the following properties:

    1.5.3.1%SegmentIteratorPrototype%.next( )

    1. Let iterator be this value.
    2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
    3. Let previousPosition be iterator.[[SegmentIteratorPosition]].
    4. Let done be AdvanceSegmentIterator(iterator, forwards).
    5. If done is true, return CreateIterResultObject(undefined, true).
    6. Let newPosition be iterator.[[SegmentIteratorPosition]].
    7. Let string be iterator.[[SegmentIteratorString]].
    8. Let segment be the substring of string from previousPosition to newPosition, inclusive of previousPosition and exclusive of newPosition.
    9. Let breakType be iterator.[[SegmentIteratorBreakType]].
    10. Let result be ! ObjectCreate(%ObjectPrototype%).
    11. Perform ! CreateDataProperty(result "segment", segment).
    12. Perform ! CreateDataProperty(result, "breakType", breakType).
    13. Perform ! CreateDataProperty(result, "position", newPosition).
    14. Return CreateIterResultObject(result, false).

    1.5.3.2%SegmentIteratorPrototype%.following( [ from ] )

    1. Let iterator be this value.
    2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
    3. If from is not undefined,
      1. Let from be ? ToIndex(from).
      2. Let length be the length of iterator.[[SegmentIteratorString]].
      3. If from ≥ length, throw a RangeError exception.
      4. Let iterator.[[SegmentIteratorPosition]] be from.
    4. return AdvanceSegmentIterator(iterator, forwards).

    1.5.3.3%SegmentIteratorPrototype%.preceding( [ from ] )

    1. Let iterator be this value.
    2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
    3. If from is not undefined,
      1. Let from be ? ToIndex(from).
      2. Let length be the length of iterator.[[SegmentIteratorString]].
      3. If from > length or from = 0, throw a RangeError exception.
      4. Let iterator.[[SegmentIteratorPosition]] be from.
    4. return AdvanceSegmentIterator(iterator, backwards).

    1.5.3.4get %SegmentIteratorPrototype%.position

    1. Let iterator be this value.
    2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
    3. Return iterator.[[SegmentIteratorPosition]].

    1.5.3.5get %SegmentIteratorPrototype%.breakType

    1. Let iterator be this value.
    2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
    3. Return iterator.[[SegmentIteratorBreakType]].
